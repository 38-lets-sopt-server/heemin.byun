# 📋 게시글 기능 명세서

---

## 1. 게시글 작성

- **요청:** 게시글 제목, 게시글 내용, 익명 여부, 첨부파일(이미지/파일)
- **저장:** 제목, 내용, 작성자 ID(익명 여부 포함), 첨부파일 S3 URL, 게시판 ID, 작성 시각
- **응답:** 생성된 게시글 ID, 저장 성공 여부
- **검증:**
    - 제목 글자수 
    - 내용 글자수 
    - 첨부파일 형식 및 용량 제한 
    - S3 업로드 성공 여부 검증 후 DB 저장
    - 게시판 ID 유효성 (존재하는 게시판인지)
    - 로그인 사용자인지 인증 확인

---

## 2. 게시글 목록 조회

- **요청:** 게시판 ID, 페이지 번호, 정렬 기준(최신순/인기순)
- **저장:** 없음 (조회만)
- **응답:** 게시글 ID, 제목, 내용 미리보기, 작성자(익명/닉네임), 작성 시각, 공감 수, 댓글 수
- **검증:**
    - 게시판 ID 유효성
    - 페이지 번호 범위

---

## 3. 게시글 상세 조회

- **요청:** 게시글 ID
- **저장:** 없음 (조회만)
- **응답:** 게시글 ID, 제목, 내용, 작성자(익명/닉네임), 작성 시각, 공감 수, 댓글 수, 댓글 리스트(대댓글 포함), 첨부파일 URL, 스크랩 여부, 본인 글 여부
- **검증:**
    - 게시글 ID 유효성 (존재하는 게시글인지)
    - 삭제된 게시글 접근 방지

---

## 4. 댓글 작성

- **요청:** 게시글 ID, 댓글 내용, 익명 여부, 부모 댓글 ID (대댓글인 경우)
- **저장:** 게시글 ID, 작성자 ID, 댓글 내용, 익명 여부, 부모 댓글 ID, 작성 시각
- **응답:** 생성된 댓글 ID, 저장 성공 여부
- **검증:**
    - 댓글 내용 글자수 
    - 게시글 ID 유효성
    - 부모 댓글 ID 유효성 (대댓글인 경우)
    - 로그인 사용자 인증 확인

---

## 5. 게시글 공감 (좋아요)

- **요청:** 게시글 ID
- **저장:** 사용자 ID, 게시글 ID, 공감 시각
- **응답:** 현재 공감 수, 공감 상태(공감함/취소)
- **검증:**
    - 중복 공감 방지 (1인 1공감)
    - 본인 게시글 공감 허용 여부 정책 결정
    - 게시글 존재 여부

---

## 6. 게시글 스크랩

- **요청:** 게시글 ID
- **저장:** 사용자 ID, 게시글 ID, 스크랩 시각
- **응답:** 스크랩 성공 여부, 현재 스크랩 상태(스크랩함/취소)
- **검증:**
    - 중복 스크랩 방지 (이미 스크랩한 경우 취소 처리)
    - 게시글 존재 여부
    - 로그인 사용자 인증 확인
